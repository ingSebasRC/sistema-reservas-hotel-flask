# Sistema de Reservas de Hotel 

Aplicación web desarrollada con Flask que permite realizar reservas de hotel
según la cantidad de huéspedes, número de días y disponibilidad.

## Funcionalidades
- Visualización de hoteles
- Selección según número de huéspedes
- Cálculo de reservas por días
- Confirmación de reserva
- Conexión a base de datos MySQL

## Tecnologías utilizadas
- Python
- Flask
- MySQL (XAMPP)
- HTML
- CSS
- JavaScript

## Instalación y ejecución
1. Clonar el repositorio
2. Crear un entorno virtual
3. Instalar dependencias
4. Configurar la base de datos MySQL
5. Ejecutar:
```bash
python app.py
